The ``1-my_list`` module
======================

Using ``print_sorted``
-------------------


This is a test file for the function print_sorted which prints a list,
but sorted (ascending sort).

    >>> MyList = __import__('1-my_list').MyList
    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.append(-5)
    >>> my_list.print_sorted()
    [-5, 1, 2, 3, 4, 5]
    >>> my_list.append(0)
    >>> my_list.print_sorted()
    [-5, 0, 1, 2, 3, 4, 5]
